# Background Remover - MVP 需求文档

## 1. 项目概述

**产品名称：** Background Remover（背景移除工具）

**产品定位：** 一个简单易用的在线图片背景移除工具，帮助用户快速去除图片背景。

**目标用户：**
- 电商卖家（需要处理商品图片）
- 设计师（快速抠图需求）
- 自媒体创作者（制作素材）
- 普通用户（证件照、头像处理）

**核心价值：** 无需专业技能，一键去除图片背景，节省时间成本。

---

## 2. MVP 核心功能

### 2.1 图片上传
- 支持拖拽上传
- 支持点击选择文件
- 支持格式：JPG、PNG、WebP
- 单次上传限制：10MB
- 显示文件名和大小

### 2.2 背景移除
- 调用 Remove.bg API 处理
- 显示处理进度/加载状态
- 处理失败时显示错误提示
- 处理时间：3-10秒

### 2.3 结果展示
- 左右对比显示（原图 vs 处理后）
- 支持放大查看细节
- 透明背景以棋盘格显示

### 2.4 图片下载
- 一键下载 PNG 格式
- 文件名：原文件名-no-bg.png
- 保留原图分辨率

---

## 3. 技术架构

### 3.1 前端
- **框架：** Next.js 15 (App Router)
- **语言：** TypeScript
- **样式：** Tailwind CSS
- **部署：** Vercel

### 3.2 后端
- **API：** Next.js API Routes
- **第三方服务：** Remove.bg API
- **存储：** 无需存储（纯内存处理）

### 3.3 基础设施
- **CDN：** Vercel Edge Network
- **域名：** 自定义域名（可选）
- **SSL：** 自动 HTTPS

---

## 4. 用户流程

```
1. 用户访问网站
   ↓
2. 上传图片（拖拽或点击）
   ↓
3. 点击"移除背景"按钮
   ↓
4. 等待处理（显示加载状态）
   ↓
5. 查看对比结果
   ↓
6. 下载处理后的图片
```

## 5. 界面设计要求

### 5.1 首页
- 简洁的标题和说明
- 明显的上传区域（虚线边框）
- 大按钮（移除背景）
- 响应式设计（支持手机）

### 5.2 配色方案
- 主色：蓝色系（#2563EB）
- 辅助色：绿色（下载按钮）
- 背景：渐变色（蓝色到紫色）
- 文字：深灰色（#1F2937）

### 5.3 交互反馈
- 上传区域 hover 效果
- 按钮 hover 变色
- 加载时显示动画
- 成功/失败提示

## 8. API 限制

**Remove.bg 免费版：**
- 每月 50 次免费调用
- 单张图片最大 12MP
- 处理时间约 5 秒

**应对方案：**
- 前端显示剩余次数（可选）
- 超限时提示用户
- 后续可升级付费版
